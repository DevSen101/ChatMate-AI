# 🤖 ChatMate AI

ChatMate AI is a lightweight Generative AI chatbot built using transformer-based language models. It generates human-like responses in real time and provides an interactive chat interface using Gradio.

---

## 🚀 Features
- 💬 Real-time conversational AI
- 🧠 Transformer-based text generation
- ⚡ GPU acceleration support (CUDA)
- 🎨 Interactive UI with Gradio
- 🧩 Simple and beginner-friendly implementation

---

## 🛠️ Tech Stack
- Transformers (Hugging Face)
- PyTorch
- LangChain
- Gradio

## ⚙️ Setup & Installation

### 1. Clone Repository
```

git clone https://github.com/DevSen101/ChatMate-AI.git
cd ChatMate-AI

```

### 2. Install Dependencies
```

pip install transformers langchain torch gradio huggingface_hub

```

### 3. Run the Notebook
- Open in Google Colab or Jupyter Notebook  
- Run all cells step by step  

---
